PeaceClock — confirmed casualty counts for active war theaters

PeaceClock presents side-by-side casualty figures with clear provenance. Every number is tied to evidence you can inspect: official reports, corroborated OSINT, and AI-assisted review — never anonymous totals.

WHAT YOU GET
• Counter view — date scrubber, authentication-threshold slider, and a side × time-window matrix (civilian figures primary, military secondary)
• Map view — clustered pins from the same backend GeoJSON as the web app, with tier and side encoding
• Source attribution — tap any figure to see linked evidence; graphic media is never embedded, only opened at the source
• Offline last-known-good — cached counts and map state with a clear "as of" timestamp when the network is unavailable

METHODOLOGY
Counts are derived from an append-only evidence store. Belligerent self-claims are excluded from confirmation. Thresholds let you see only official, confirmed, OSINT, or AI-corroborated tiers.

PeaceClock is informational, neutral, and non-partisan. No accounts, no ads, no purchases.
